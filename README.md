# Proyecto Programación Orientada a Objetos en Python
 Descripción del Proyecto

Este proyecto es una aplicación desarrollada en Python que implementa los principales conceptos de la Programación Orientada a Objetos (POO), tales como herencia, encapsulación y polimorfismo.

El programa simula la gestión de animales, utilizando una clase base y una clase derivada, y está organizado en carpetas para separar responsabilidades.

# Conceptos de POO Implementados

Herencia:
La clase Perro hereda de la clase base Animal.

Encapsulación:
Los atributos nombre y edad de la clase Animal están protegidos y se accede a ellos mediante métodos.

Polimorfismo:
El método hacer_sonido() es sobrescrito en la clase Perro, permitiendo comportamientos diferentes para objetos relacionados.
# Ejecución del Programa

Para ejecutar el programa, asegúrese de tener Python instalado y ejecute el siguiente comando:
python main.py
